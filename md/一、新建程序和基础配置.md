## Django程序 gzjblog

### 1、虚拟环境中创建项目gzjblog和应用article

```python 
# 查看虚拟环境
G:\django-blog>lsvirtualenv
# 进入虚拟环境
G:\django-blog>workon django_2.2
# 虚拟环境中创建项目gzjblog
(django_2.2) G:\django-blog>django-admin startproject gzjblog
# 进入项目目录并创建应用article
(django_2.2) G:\django-blog\gzjblog>python manage.py startapp article
```

### 2、pycharm打开项目并做如下配置

> 配置应用

```python
my_blog/settings.py
#配置应用
INSTALLED_APPS = [
	...
    'article',
]
```

配置应用后项目就可以启动并运行了，默认的django页面

> 配置url

```python
# 主路由配置
from django.contrib import admin
# 新增inclue引用
from django.urls import path,include
urlpatterns = [
    ...
    # 分发到分路由
    path('article/', include('article.urls',namespace='article')),
]

# 分路由配置-article中新建urls.py
from django.urls import path
from . import views
app_name='article'

urlpatterns=[
    path('article_list/',views.list,name='article')
]
```

> 配置模板文件路径

```python
# 项目下新建文件夹templates并做如下配置
TEMPLATES = [
    {
        ...
        # 模板文件路径配置
        'DIRS': [os.path.join(BASE_DIR,'templates')],
		...
    },
]
```

> 静态文件夹配置

```python
# 项目下新建文件夹static并做如下配置
# 建项目时已有
STATIC_URL = '/static/'
# 增加配置
STATICFILES_DIRS=[
    os.path.join(BASE_DIR,'static')
] 
```

> 数据库配置

```python
# 1.安装pymsql
pip install pymysql
# 2.gzjblog/__init__.py中
import pymysql
pymysql.install_as_MySQLdb()
# 3.gzjblog/settings中(前提是先建立数据库)
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',  # 数据库引擎
        'NAME': 'gzjblog',  # 你要存储数据的库名，事先要创建之
        'USER':'root',    # 数据库用户名
        'PASSWORD':'123456', # 密码
        'HOST':'127.0.0.1',  # 主机
        'PORT':'3306',       # 数据库使用的端口
    }
}
# 新建mysql数据库
# 进入数据库
mysql -uroot -p123456
# 新建gzjblog
create database gzjblog charset utf8;
```
